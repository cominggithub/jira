About

This is a Flask web application with SQLAlchemy and multiple database support, featuring a comprehensive theme system.

Features

  • SQLAlchemy Integration: Full ORM support with multiple database configurations
  • Environment-based Configuration: Separate configs for development, production, and testing
  • Multiple Database Support: Primary, analytics, and cache database configurations
  • Theme System: Four distinct visual themes (Neon, Tron, Dark, White)
  • Responsive Design: Mobile-friendly layouts across all themes
  • Modern CSS: CSS Grid, Flexbox, and CSS custom properties

Technology Stack

  • Backend: Flask, SQLAlchemy, Python-dotenv
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Database: SQLite (development), PostgreSQL (production ready)
  • Styling: CSS custom properties, CSS animations
